 # Encuestas
 

### What was used on this project

- ReactJs
- Nodejs
- Express
- MongoDB

## Deploy Heroku

```
https://shatdow-encuestas.herokuapp.com/login
```
### Clone the Repo
```
https://github.com/Moises6669/Encuestas.git
```

### Install the dependencies

```
yarn install -> Reactjs
```

```
Npm install -> Nodejs
```

### Run App 

```
yarn start
```

### Run the server

```
Npm run dev
```
